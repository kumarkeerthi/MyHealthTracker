# Exercise Engine

The exercise engine scores movement quality and consistency while supporting multiple categories:

- WALK
- BODYWEIGHT
- MONKEY_BAR
- STRENGTH

## Key outputs
- Strength score
- Grip improvement %
- Monkey-bar progression index
- Weekly strength trend graph

## Rule integrations
- Post-meal walk bonus for qualifying walks.
- Movement type validation by category.
- Signals feed metabolic advisor recommendations.
